このブログを運営している Paratoki Lab について、簡単に自己紹介させてください。

「個人で作るアプリは、必要な人にだけそっと届けば充分」 — そんなスタンスで、日々の暮らしにささやかに寄り添うアプリを開発しています。便利機能を全部詰め込むのではなく、**プライバシーと利便性のバランス**を意識した作りを心がけています。

## 公開しているアプリ

### アニあと！〜私のアニメの軌跡〜

アニメの視聴記録を端末ローカルに保存できる Android アプリです。クラウド同期はせず、データはすべて自分の手元に残ります。バックアップも自分で取れる範囲で。

「全部クラウドで自動」は便利ですが、データを誰かに預ける覚悟も必要です。アニメの視聴履歴ぐらい、自分だけが知っていればいいかな、という発想で作りました。

### チックンノート（LINE ミニアプリ）

お子さまの予防接種スケジュールを LINE で管理できる、LIFF（LINE Front-end Framework）ベースのミニアプリです。

母子手帳の予防接種ページの代わりとして、保護者の手元にスケジュールと記録が残るように設計しています。専用アプリのインストール不要、LINE 上から直接アクセスできます。次回接種の前日には LINE でリマインダーが届く仕組みです。

## このブログで書くこと

主に以下のテーマで記事を書いていきます。

- **個人開発の振り返り** — どんなアプリを、どんな思いで作ったか
- **技術メモ** — Next.js / Astro / Firebase / LINE LIFF / Cloud Functions など、実際に詰まった点と解決方法
- **個人運営のノウハウ** — 月額 0 円〜数百円で運用するためのアーキテクチャ判断、規約・プライバシーポリシーの整え方など

中でも、チックンノートを **個人開発で月額 0 円で立ち上げた一連の記録**を「LINE ミニアプリ開発記」として連載しています。構想から実装、ハマりポイント、運用まで、フィクションを混ぜずに自分が踏んだ手順をそのまま残しています。
